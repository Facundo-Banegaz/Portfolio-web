Api Disney

Una Web API construida con C# para gestionar información sobre películas y personajes de Disney. Permite:

  • Autenticación y Autorización: Registro, login y protección de rutas con JWT según roles de usuario.
  • CRUD de Personajes y Películas: Operaciones de crear, leer, actualizar y eliminar, solo para usuarios autorizados.
  • Listados y Detalles: Ver personajes y películas con detalles, algunos accesos requieren autorización.
  • Búsqueda y Filtros: Buscar y filtrar personajes y películas, con rutas protegidas.

E-commerce de Suplementos

Es un ecommerce desarrollado con ASP.NET Core, donde los usuarios pueden comprar suplementos en línea con funcionalidades de búsqueda avanzada, filtros, paginación, carrito de compras, login, registro de usuarios y sección de administradores con dashboard. La autenticación y autorización se gestionan a través del correo electrónico. Utilicé Entity Framework para la gestión de la base de datos. Aún está en fases finales de desarrollo, pero ya cuenta con la mayoría de las funcionalidades implementadas.
